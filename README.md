# Smart Solar Wireless EV Charging Station with IoT Monitoring

## Overview
An ESP32-based smart EV charging prototype powered by solar energy with wireless power transfer and IoT monitoring.

## Features
- Solar-powered charging
- Wireless power transfer
- Vehicle detection using IR sensor
- Voltage and current monitoring
- Blynk IoT dashboard

## Components Used
- ESP32
- IR Sensor
- LDR Sensor
- Voltage Sensor
- Current Sensor
- Relay Module
- Wireless Charging Module

## Technologies
ESP32, Arduino IDE, Blynk IoT
